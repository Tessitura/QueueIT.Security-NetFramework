WelcomeQueueIT.Security Documentation

Welcome to the Queue-it Known User implementation Guide. This is intended to be a technical reference that you can use to find out all that you need to know to implement Queue-it Known User request validation in your project. If you have any information, tips, updates, or corrections that you would like to see added to the guide, feel free to submit them to the author using the Send Feedback link in the page footer.

This documentation is structured as follows

Getting started

Here you can read about our different implementation strategies based on a technology choice, e.g. ASP Web Forms, ASP MVC, PHP or Java EE. We currently support 2 different implementation strategies. The recommended one is ‘Session’ but if this doesn’t align well with your current situation, please be free to go with the ‘Link’ strategy instead.
